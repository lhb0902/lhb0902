Created2023-03-28|anaconda|anaconda常用指令
1.conda指令创建名为name的虚拟环境，同时安装python3.6 $ conda create -n name python=3.6 2.查看已安装虚拟环境 $ conda env list 3.激活虚拟环境 $ conda activate name $ source activate name 4.退出虚拟环境 $ conda deactivate $ source deactivate 5.删除虚拟环境 $ conda remove -n name –all 6.创建虚拟环境 $ conda create -n env_name list_of_packages 7.分享虚拟环境，将当前环境下的package信息存入名为environment的YAML文件中 $ conda env export > environment.yaml 8.使用别人生成的yaml文件创建虚拟环境 $ conda env create -f environment.yaml
Hexo框架使用
Created2023-02-23|Hexo框架|Hexo框架的使用
本文中记录了Hexo框架的常用指令及其简写，在部署Hexo框架之间需要进行npm、git等相关工具的安装，详细的框架部署过程见https://blog.csdn.net/weixin_46415275/article/details/124720940 。 1.初始化网站根目录 $ hexo init 2.创建新文章 $ hexo new “my_title” 3.清理缓存文件 $ hexo clean 4.生成静态文件 $ hexo generate 可简写为 hexo g 5.启用本地服务器 $ hexo server 可简写为 hexo s 6.部署网站 $ hexo deploy 可简写为 hexo d 7.进入调试模式(将消息详细记录到终端和debug.log文件) $ hexo –debug
