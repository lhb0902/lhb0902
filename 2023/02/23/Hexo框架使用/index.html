Hexo框架使用

2023-02-23

&emsp; &emsp; 本文中记录了Hexo框架的常用指令及其简写，在部署Hexo框架之间需要进行npm、git等相关工具的安装，详细的框架部署过程见https://blog.csdn.net/weixin_46415275/article/details/124720940 。

1.初始化网站根目录

$ hexo init

2.创建新文章

$ hexo new “my_title”

3.清理缓存文件

$ hexo clean

4.生成静态文件

$ hexo generate 可简写为 hexo g

5.启用本地服务器

$ hexo server 可简写为 hexo s

6.部署网站

$ hexo deploy 可简写为 hexo d

7.进入调试模式(将消息详细记录到终端和debug.log文件)

$ hexo –debug
