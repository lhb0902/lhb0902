1. 什么是Portainer

Portainer 是一款轻量级的应用，它提供了图形化界面，用于方便地管理Docker环境，包括单机环境和集群环境。

2. 基础安装

1. 拉取最新的Portainer

2. 安装和启动

  • docker run: 运行容器的命令。
  • -d: 这个标志表示容器应该以分离模式运行，意味着它在后台运行，不会阻塞命令行。
  • --restart=always: 此标志确保容器在停止时始终重新启动，除非用户明确停止它。
  • --name portainer: 此标志为容器指定一个特定的名称，在这种情况下为 “portainer”。
  • -p 9000:9000: 此标志将主机的9000端口绑定到容器的9000端口。这允许主机访问容器的9000端口。
  • -v /var/run/docker.sock:/var/run/docker.sock: 此标志将主机的 Docker socket 挂载到容器中。这允许容器与主机上的 Docker 守护程序进行通信，这是一种与 Docker 交互的常见方法。
  • portainer/portainer: 这指定了要创建容器的镜像，这种情况下使用的是 portainer/portainer 镜像。

3. 访问 Portainer

  • 地址：**http://localhost:9000/**
  • 操作：登录后设置你的用户名和密码，并设置本地Docker即可
